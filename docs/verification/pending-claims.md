# 內容補寫時引入的待查證敘述（2026-07-18）

## income-tax
- [ ] 健保卡＋密碼登入報稅系統的密碼設定／重設方式，以財政部電子申報繳稅服務網站公告為準
- [ ] 扶養親屬認定的所得與年齡門檻，以國稅局當年度公告為準
- [ ] 退稅入帳與補稅繳納的實際期限，以財政部公告為準

## house-tax
- [ ] 自住適用的全國房屋持有戶數上限，以財政部／地方稅稽徵機關當年度公告為準
- [ ] 自住稅率申請後的回溯適用期間，以受理機關公告為準
- [ ] 審核所需作業天數，以房屋所在地地方稅稽徵機關公告為準

## gift-tax
- [ ] 贈與稅逾期申報的滯報金／罰鍰計算方式，以國稅局公告為準
- [ ] 贈與稅分段稅率與級距金額，以財政部當年度公告為準
- [ ] 不動產贈與是否須先繳清稅款才能取得過戶用證明，以受理國稅局說明為準

## vehicle-tax
- [ ] 牌照稅／燃料費逾期滯納金計算方式，以地方稅稽徵機關及監理機關公告為準
- [ ] 牌照稅分期繳納是否受理及申請條件，以車籍所在地地方稅稽徵機關規定為準

## license-renew
- [ ] 線上申請補發駕照郵寄所需天數，以監理服務網公告為準
- [ ] 駕照換發是否需要現場拍照或可自備照片，以各監理所／站規定為準

## ticket
- [ ] 陳述意見（申訴）被駁回後的裁罰標準是否可能加重，以處罰機關規定為準
- [ ] 逾期未於應到案期限處理的加徵滯納金計算方式，以處罰機關公告為準
- [ ] 陳述意見／申訴的受理方式（線上、郵寄或需親自到場）依個案與舉發機關規定，送件前先電話確認

## land-transfer
- [ ] 契稅、土地增值稅由買賣哪一方負擔及各自的申報／繳納期限，以稅捐機關公告及契約約定為準
- [ ] 地政事務所送件後補正期限與駁回規定，以受理地政事務所通知為準
- [ ] 自辦過戶與委託地政士辦理各自實際所需工作天數，可能因案件複雜度而有差異

## scrap-car
- [ ] 報廢登記與地方稅稽徵機關稅籍停徵之間的作業時間差，以各監理站及地方稅稽徵機關實務為準
- [ ] 車輛設有動產抵押（車貸）時報廢是否需先徵得貸款機構同意，以監理站受理規定為準
- [ ] 廢車回收獎勵金實際核發流程與所需文件，以環保單位／監理站公告為準

## childcare-subsidy
- [ ] 育兒津貼／托育補助是否有定期複核機制及頻率，各縣市規定需查證
- [ ] 審核核定所需時間因縣市而異，需電話向戶籍地公所確認

## low-income
- [ ] 是否安排家訪、家訪頻率因公所而異，需查證
- [ ] 每年底重新調查的確切時間與方式需向戶籍地公所確認

## disability
- [ ] 複檢／申訴的期限與流程需向核發公所或衛福部查證
- [ ] 證明有效期限（永久 vs 定期）依障礙類別而異，需查證

## elderly
- [ ] 敬老卡是否需本人親辦或可代辦，依各縣市公所規定不同，需查證
- [ ] 長照部分負擔具體金額與分級標準以衛福部最新公告為準

## national-pension
- [ ] 國保自動納保／停保的具體觸發條件與時點需以勞保局公告為準
- [ ] 減免／緩繳的所得標準與申請條件需查證勞保局最新公告

## labor-dispute
- [ ] 資遣費計算公式與年資對照需以勞動部公告為準，不自行推算具體金額
- [ ] 欠薪墊償等求償機制的適用條件與請領方式需向勞動部或勞保局查證

## labor-pension
- [ ] 不同出生年度對應的老年給付請領年齡與年資門檻，需以勞保局最新公告為準
- [ ] 勞保與國保年資合併計算的具體規則需向勞保局查證

## nhi-card
- [ ] 代辦補發（監護人／家屬）所需的證明文件以健保署最新公告為準，需電話或官網確認

## name-change
- [ ] 一生改名次數上限與例外情形（未成年、特殊事由等）的具體條件，以戶政事務所最新公告及個案認定為準

## divorce
- [ ] 未成年子女親權書寫格式與戶所實際要求細節，以戶政事務所現場範本及承辦認定為準

## arc
- [ ] ARC 延期可送件的確切起算時點（屆滿前 3 個月的計算方式）
- [ ] 依親居留所需戶籍／關係證明文件的實際受理版本
- [ ] 居留期間工作異動申報的具體規定

## conscription-exit
- [ ] 役男出境核准的確切效期與是否需按次重新申請
- [ ] 護照或身分資料不符時的具體退件條件

## police-record
- [ ] 不同用途（移民／簽證／求職）良民證版本與記載內容的具體差異
- [ ] 涉及司法查詢案件實際核發所需天數
- [ ] 線上申辦所需身分驗證工具種類

## lost-found
- [ ] 撿到遺失物未送交的法律責任與期限規定
- [ ] 交通工具（計程車／捷運／客運）遺失物招領流程與轉交警方的時間

## certificate-auth
- [ ] 法院公證處與民間公證人在費用結構、辦理時間上的具體差異
- [ ] 國外用途文件的格式要求依對象國家而定的細節
- [ ] 當事人無法親自到場之替代辦理方式與適用條件
