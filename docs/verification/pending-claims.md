# 內容補寫時引入的待查證敘述（2026-07-18）

## land-transfer
- [ ] 自辦過戶與委託地政士辦理各自實際所需工作天數，可能因案件複雜度而有差異

## childcare-subsidy
- [ ] 育兒津貼／托育補助是否有定期複核機制及頻率，各縣市規定需查證
- [ ] 審核核定所需時間因縣市而異，需電話向戶籍地公所確認

## low-income
- [ ] 是否安排家訪、家訪頻率因公所而異，需查證
- [ ] 每年底重新調查的確切時間與方式需向戶籍地公所確認

## disability
- [ ] 複檢／申訴的期限與流程需向核發公所或衛福部查證

## national-pension
- [ ] 緩繳（非「所得未達一定標準」減免以外的其他緩繳條件）的具體規則需查證勞保局最新公告

## divorce
- [ ] 未成年子女親權書寫格式與戶所實際要求細節，以戶政事務所現場範本及承辦認定為準（本輪已補上戶政司官方參考範本連結，格式細節仍以現場承辦認定為準）

## arc
- [ ] ARC 延期可送件的確切起算時點（屆滿前 3 個月的計算方式）
- [ ] 依親居留所需戶籍／關係證明文件的實際受理版本
- [ ] 居留期間工作異動申報的具體規定

## conscription-exit
- [ ] 役男出境核准的確切效期與是否需按次重新申請
- [ ] 護照或身分資料不符時的具體退件條件

## police-record
- [ ] 不同用途（移民／簽證／求職）良民證版本與記載內容的具體差異
- [ ] 涉及司法查詢案件實際核發所需天數
- [ ] 線上申辦所需身分驗證工具種類

## lost-found
- [ ] 撿到遺失物未送交的法律責任與期限規定
- [ ] 交通工具（計程車／捷運／客運）遺失物招領流程與轉交警方的時間

## certificate-auth
- [ ] 法院公證處與民間公證人在費用結構、辦理時間上的具體差異
- [ ] 國外用途文件的格式要求依對象國家而定的細節
- [ ] 當事人無法親自到場之替代辦理方式與適用條件
