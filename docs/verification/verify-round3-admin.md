> 執行這批查證的子代理在完成前 5 個業務後，因 API session 額度限制被系統中止，未及寫出逐條查證細節就結束。以下依實際套用到 `js/data.js` 的修改內容回填記錄。

## 已完成（verified: 2026-07-18）

- **license-renew（駕照換發／補發）**：補正「監理站原則上不提供現場拍照，需自備符合規格照片」；線上補發郵寄時間補上「約 7–10 個工作天」，臨櫃約 15–30 分鐘可領。
- **ticket（交通罰單申訴與繳納）**：修正「申訴被駁回是否加重」敘述——原文誤植為「原則不會加重」，改為「結果可能維持原處分、撤銷、減輕，也可能依裁罰基準表加重，並非一律加重」；補上逾期未到案「加徵滯納金依逾期天數分級遞增」；補上陳述意見受理方式（親自到場／郵寄／線上／傳真，依個案與機關規定）。
- **scrap-car（汽機車報廢）**：牌照稅籍停徵修正為「原則自次月起停止課徵，機關間偶有作業時間差」；車貸／動保設定報廢需先取得貸款機構同意或塗銷動保；廢車回收獎勵金補上完整申請流程（「車輛回收服務一站通」線上申請免附文件、約 3 週入帳、回收日次日起 5 年內申請）。
- **name-change（改名／改姓）**：3 次上限補上法源（姓名條例第 9 條第 1 項第 6 款）與例外情形（與尊親屬同名、被收養等不計入 3 次；未成年改過一次須成年後才能再改）。
- **divorce（離婚登記）**：未做內容修正（親權書寫格式仍因戶所而異，查無統一官方規定），但補上戶政司官方「兩願離婚協議書參考範本」PDF 連結供參考。此項在 pending-claims.md 保留待查證標記。
