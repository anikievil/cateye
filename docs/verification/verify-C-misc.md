## 5. 失業給付（`unemployment`，data.js line 421-427）

**原值**：非自願離職、約前6個月平均投保薪資60%、最長6個月（45歲以上或身障9個月）；申請後14日推介就業期；每月失業再認定需2次以上求職紀錄；請求權2年

**結論：全部確認**
- 非自願離職、平均月投保薪資60%（有眷屬加給每人10%，最多20%）→ **確認**
- 最長6個月，45歲以上或領有身心障礙證明者最長9個月 → **確認**
- 求職登記後14日內無法推介就業/安排職訓才完成失業認定（開始核發）→ **確認**

- 申請期限（請求權）：非自願離職退保翌日起2年內申請失業（再）認定；續領者自前次領取期間末日翌日起2年內申請 → **確認**
